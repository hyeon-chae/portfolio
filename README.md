# HYEON PORTFOLIO

프론트엔드 개발자 채현의 포트폴리오 웹사이트입니다.

## 기술 스택

- HTML5
- CSS3 / SCSS
- Vanilla JavaScript

## 프로젝트 실행

```bash
# SCSS 컴파일 (watch 모드)
npm run sass
```

## 배포

GitHub Pages를 통해 배포됩니다.

- URL: https://hyeon-chae.github.io/portfolio/

## 주요 기능

- 반응형 디자인 (모바일/데스크톱)
- 스크롤 기반 네비게이션
- 프로젝트 모달 비디오 플레이어
- 자동 업데이트 날짜 표시
